# Item-Based Collaborative Filtering – Reproduction

This repository reproduces Sarwar et al. (WWW10 2001) using MovieLens 100K.

## How to run

1. Install dependencies: `pip install -r requirements.txt`
2. Place `u.data` in `data/ml-100k/`
3. Run the notebook `src/your-notebook.ipynb`
